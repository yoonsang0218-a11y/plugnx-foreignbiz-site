### 2-2. 포지셔닝 2×2 — 두 축 중 한 축만 채워진 시장

- **X축 = 양면 통합도** (단변 single-side → 양면 B2B↔B2C 한 데이터 파이프라인)
- **Y축 = 인증·데이터 인프라 깊이** (중개·앱 레벨 → 자체 본인인증 인프라 + 대안신용 데이터 결합)

```
인증·데이터          ↑ 인프라 깊음
인프라 깊이          │
                    │  [좌상] 자블리            ┌──────────────────────────┐
                    │  B2B 비자판정 기술은      │ [우상 = 빈 공간 = PlugNX] │
                    │  깊으나 단변              │ 양면 통합 × ForeignID    │
                    │  (B2C·환급·금융 없음)    │ (L1~L3) + 고용↔금융      │
                    │                          │ 데이터 → 외국인 신용스코어│
                    │                          └──────────────────────────┘
   ─────────────────┼────────────────────────────────────────────────────→
   단변             │                                          양면 B2B↔B2C
                    │  [좌하] 송금 핀테크·삼쩜삼   [우하] 클링커즈(glow)
                    │  단품·앱 레벨               B2C 다변(생활·금융)이나
                    │  (송금/세금환급 only)       B2B·인증 인프라 부재
                    │
                    ↓ 인프라 얕음 (중개·앱)
```

- **좌하단(단변·앱)**: 송금 핀테크(센트비·한패스), 삼쩜삼 — 한 기능만, 인증은 외부 의존.
- **우하단(B2C 양면적이나 인프라 얕음)**: 클링커즈 glow — B2C 다변이나 자체 KYC·B2B 고용 데이터 미확인.
- **좌상단(인프라 깊으나 단변)**: 자블리 — B2B 비자판정 기술은 깊으나 B2C·환급·플라이휠 없음.
- **우상단(빈 공간)**: PlugNX — 양면 통합 × 인증 인프라+신용스코어. **공개 출처상 이 좌표를 점유한 사업자 미확인**(="유일"이 아니라 "미확인 공백"으로 한정 표기).

캡션: **경쟁사는 두 축 중 한 축만 채운다 — 우상단은 비어 있다.** (단, 빈 공간이 곧 승리를 보장하지 않으며, 인증 인프라+양면 결합이 규제·실행상 성립하는지는 PoC·법률 검증 대상.)

### 2-3. Moat score — 모방 난이도 5점 평가 (시제 분리)

각 해자를 ⓐ 모방 난이도(5=복제 매우 어려움) ⓑ **시제(현재 보유 now / 축적 future)**로 평가. 점수는 **전략적 판단(추정)**이며 정량 검증 전이다.

| 해자 | 모방 난이도 | 시제 | 근거 / 한계 |
|---|---|---|---|
| **양면 네트워크 효과** | 5 / 5 | **보유·설계** (효과치는 future) | 제품 흐름(VisaDesk→Moneeback 초대)은 구현. 단면 경쟁사는 한쪽을 CAC 0으로 채울 입구 없음. 단, 전환 60%+는 PoC 검증 목표 가정. |
| **고용+금융 데이터 = 외국인 신용스코어** | 5 / 5 | **축적 future** (가설) | 데이터는 시간으로만 쌓임 → 후발은 과거를 못 삼. 단, 신용정보법·개인정보보호법상 합법성·대안신용평가 인가는 미검증(해자가 규제로 막히면 약화). |
| **정책 정합성(K-Trust·K-CORE·쿼터·임금자문위)** | 4 / 5 | **보유·정합** | 정부 제도에 데이터 포맷이 맞물림 → 규제 강화가 수요 증가. 도메인·관계 자본은 단기 복제 난. 단, 정책 자체 변동 리스크. |
| **전환비용·워크플로 락인** | 4 / 5 | **축적 future** | 명단·이력·서류·알림 루틴 누적 시 이탈 비용↑. 고객 0단계엔 미발생, 운영 데이터 쌓인 뒤 강화. |

| **제품 확장성(ForeignID L1~L3 인증 인프라)** | 4 / 5 | **보유·구현 중** | 단변 경쟁자가 다른 변으로 가려면 인증 인프라부터 재구축. 단, L3(계좌·전자서명) 완성도·규제 적합성은 검증 단계. |
| **∑ 복리(5겹 중첩)** | 5 / 5 | 혼합 | 네트워크→데이터→정합→락인→전문가가 서로 강화. 한 겹씩은 베껴도 복리는 못 따라옴. **단, 복리는 고객·데이터가 쌓인 뒤 작동하는 미래 명제.** |

## ③ So-what (전략적 함의)

1. **시장은 '한 축'만 채워져 있다.** 직접 경쟁사(자블리·클링커즈)는 강하지만 각자 B2B-only 또는 B2C-only다. PlugNX의 차별성은 '더 잘함'이 아니라 **'두 축을 동시에 점유하는 좌표 자체가 비어 있음'**에 있다 — 단, 이는 "유일"이 아니라 "공개 출처상 미확인 공백"으로 표기해야 방어 가능.
2. **해자의 70%는 미래 명제다.** 데이터 신용스코어·전환비용·복리는 '축적 후 발생' 가설이며, 규제(신용정보법·대안신용평가 인가) 통과가 전제다. **현재 즉시 작동하는 해자는 양면 제품 흐름·정책 정합·전문가망**이고, 이것이 PoC에서 팔리는 무기다.
3. **싸우는 곳과 피하는 곳을 명확히 분리하라.** 직접 진입(여신·송금·B2C 생활포털 전면전·채용 마켓플레이스)은 No-go이거나 제휴 우선. **싸울 전장은 '채용 이후 재직관리 OS(중소·중견 5~50명)'** 단 하나다.
4. **가장 강한 반론은 대기업 in-house와 공공 무료화.** 이 둘에 대한 답("롱테일 + 양면 + 구독·데이터 자산")을 IR·세일즈에서 선제적으로 제시해야 신뢰도가 오른다.

- **04 가치제안·가격**: '빈 공간' 차별성을 가격·패키징으로 어떻게 화폐화할지(양면 결합 가치를 ARPA에 반영). win-theme 4개를 패키지 메시지로 연결.
- **06 영업 모션**: 직접/인접 경쟁사별 'trap/언급 시 대응 스크립트' 필요(자블리=인접 연동, 클링커즈=우회). 2×2를 세일즈덱 자산화.
- **08 비즈니스 케이스**: 반격 (a)/(b) 시나리오를 리스크 시나리오·민감도에 반영. 해자의 '미래 명제' 비중을 밸류에이션 가정에 라벨.
- **01 세그멘테이션**: 세그별 win-theme(제조/요양/농축수산)와 세그 우선순위·SOM 정합 확인 필요.
- **확인 필요(리서치 후행)**: ① 자블리 매출 15.2억·정확도 75→92%·고객사(쿠팡/CJ/롯데)는 회사발표 2차정보·독립검증 안 됨. ② 클링커즈 회원수·거래액·KB 관계 규모 미확인(KB유니콘클럽 선정 수준). ③ 코워크 "연 20~30%·10곳 중 7곳"은 사업자 진술. ④ **외국인 신용스코어 양면 결합의 신용정보법·개인정보보호법 합법성·대안신용평가 인가 가능성 = 법률·금융규제 전문가 검증 필수**(해자 핵심이 규제로 막히면 우상단 차별성 약화). ⑤ 인터넷은행·은행 외국인 여신 역진입 시점.

> ⚠️ 본 분석의 경쟁사 사실은 법적 효력 없는 **참고 정보**이며 공개·언론 자료 기준이다. 인용 전 각 사 공식 자료 재확인·기준시점 명시가 필요하다. 비자·금융·신용평가 관련 판단은 행정사·변호사·금융규제 전문가 검증과 최신 법령 확인이 필요하다. '유일/보장' 단정은 쓰지 않으며, 차별성은 '공개 출처상 미확인 공백'으로 한정한다.

## ⑥ 헤드라인

**경쟁사는 양면 통합 × 인증 인프라 2축 중 한 축만 채운다 — PlugNX는 비어 있는 우상단('채용 이후 재직관리 OS')을 점유하되, 해자의 70%(데이터·락인·복리)는 PoC·규제로 검증할 미래 명제임을 라벨링해 과장 없이 차별화하라.**

**win-theme 3선:**
1. **"두 축을 동시에 채우는 좌표가 비어 있다"** — 단변 경쟁자는 다수지만 ForeignID 인증 인프라 + 고용↔금융 양면 결합 조합은 공개 출처상 미확인(유일 아님, 공백).
2. **"채용 이후 운영 공백이 우리 전장"** — 채용 전(자블리)·송금/환급(핀테크)·생활포털(클링커즈)은 피하거나 제휴, 재직관리 OS(중소·중견 5~50명)에 집중.
3. **"지금 파는 해자와 나중에 쌓는 해자를 분리"** — 현재 무기는 양면 흐름·정책 정합·전문가망, 데이터 신용스코어·락인·복리는 축적 후 검증 가설(시제 라벨로 신뢰 확보).
